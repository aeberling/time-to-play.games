# Card Images

Place card images here organized by faction.

## Structure

```
cards/
├── faction1/
│   ├── f1_attack_boost.png
│   ├── f1_defense_wall.png
│   └── ...
└── faction2/
    ├── f2_fury.png
    ├── f2_armor.png
    └── ...
```

## Specifications

- **Format**: PNG or WebP
- **Dimensions**: 300x420px (card portrait aspect ratio)
- **Color Mode**: RGB
- **Resolution**: 72 DPI minimum
- **Compression**: Optimized for web

## Card Elements

Each card should include:
- Card name (at top)
- Faction symbol/color
- Artwork/illustration
- Attribute values (attack, defense, etc.)
- Ability text
- Resource cost (if applicable)
