# UI Assets

Reusable UI elements for War in Heaven.

## Files

- `card-back.svg` - Card back design
- `hex-overlay.svg` - Hex overlay patterns
- `faction1-icon.svg` - Faction 1 icon/symbol
- `faction2-icon.svg` - Faction 2 icon/symbol
- `victory-icon.svg` - Victory point icon
- `resource-icon.svg` - Special resource icon

## Specifications

- **Format**: SVG (preferred)
- **Color**: Use CSS variables for theming when possible
- **Size**: Scalable, design at appropriate base size
