# Board Assets

Place hex tile and board background assets here.

## Files

- `hex_tile.svg` - Base hex tile
- `hex_tile_highlight.svg` - Highlighted hex (hover)
- `hex_tile_selected.svg` - Selected hex
- `hex_tile_valid_move.svg` - Valid move indicator
- `hex_tile_objective.svg` - Objective terrain
- `hex_tile_defensive.svg` - Defensive terrain
- `board_background.svg` or `.png` - Background texture/pattern

## Specifications

- **Format**: SVG for tiles, PNG/WebP for backgrounds
- **Hex Orientation**: Choose one (flat-top or pointy-top) and be consistent
- **Size**: Design at standard size (e.g., 80px for flat-to-flat)

## Hex Tile Design

- Clear borders for visibility
- Subtle terrain variations
- Highlights should be non-intrusive
- Color-blind friendly colors
