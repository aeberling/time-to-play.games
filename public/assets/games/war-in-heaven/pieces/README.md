# Game Piece Sprites

Place game piece images here organized by faction.

## Structure

```
pieces/
├── faction1/
│   ├── warrior.svg
│   ├── archer.svg
│   ├── champion.svg
│   └── ...
└── faction2/
    ├── soldier.svg
    ├── sniper.svg
    ├── berserker.svg
    └── ...
```

## Specifications

- **Format**: SVG (preferred for scalability) or PNG
- **Dimensions**: Design at 100x100px, will scale
- **Style**: Clear silhouettes for visibility on hex grid
- **Color**: Use faction colors or neutral with CSS coloring

## Design Guidelines

- Pieces should be recognizable at small sizes
- High contrast for visibility
- Distinct shapes for each piece type
- Consider variations for damaged/wounded states
